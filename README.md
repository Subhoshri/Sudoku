## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/sudoku-game.git
cd sudoku-game
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Start the Development Server
```bash
npm start
```

The app will run at `http://localhost:3000`.
